# Visualisasi Data Seismik (Streamlit)

Repository ini berisi tugas praktikum Teknik Pemrograman Minggu 13
tentang visualisasi data seismik menggunakan Streamlit dan Matplotlib.

## Fitur
- Dropdown colormap
- Auto dan Manual scaling (vmin & vmax)
- Visualisasi interaktif data seismik

## Cara Menjalankan
```bash
pip install streamlit matplotlib numpy
streamlit run minggu13.py
